# Veo, Veo - Artisan Cookies

A beautiful static website for an artisan cookie shop.

## About

Veo, Veo is a demo website showcasing artisan cookies with a modern, responsive design. Features include:

- Product catalog with shopping cart
- Recipe section
- About us page
- Mobile-responsive design
- Beautiful UI with Tailwind CSS

## Live Demo

Visit the live site at: https://[your-username].github.io/veo-veo

## Technologies

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Swiper.js for carousels
- Lucide Icons

## Local Development

Simply open `index.html` in your browser. No build step required!
